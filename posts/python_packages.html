ROSSpy

  • An API and UI that simulates the geochemical reactive transport of Reverse Osmosis desalation.
  • The module importantly predicts mineral precipitation, which causes scaling and module destruction

Codons

  • An API conducts the central dogma of biology and executes BLAST results for protein and genetic sequences.

PDIpy

  • An API and UI that predicts the log-reduction of a bacterial solution or biofilm colony from Photodynamic inactivation.
  • The module applies chemical kinetics to fundamentally model membrane oxidation over the course of time, and with the parameterized experimental conditions.

ChemW

  • An API that precisely calculates the MW from any chemical formula, including inorganic complexations and decimal stoichiometry.
  • The module further offers the ability to refine PHREEQC databases into programmable JSON files.
  • The aforementioned use-cases of ChemW were applied in the ROSSpy module

Hillfit

  • An API that calculates regression according to the biochemical Hill-equation for a set of data.
  • The module exports a regression plot with the R2, a TXT file of the equation and parameter values, and provides API access to all components of the equation.
  • HillFit was critically applied in refining the predictions of the PDIpy module
